Google Sheets sync
Paste your Apps Script Web App URL here. Every time you save a day it syncs automatically.
Setup steps:
1. Go to sheets.google.com → create a new sheet
2. Click Extensions → Apps Script
3. Delete existing code, paste in Code.gs from your repo
4. Click Deploy → New deployment
5. Type: Web app — Execute as: Me — Who has access: Anyone
6. Click Deploy → copy the Web App URL → paste above
